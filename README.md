# Aplicativo de orçamento pessoal e investimentos com o framework Django

Esta é a maneira como eu monitoro e controlo minhas despesas diariamente. Desenvolvido do zero e originalmente um estudo sobre como construir coisas com o framework Django, este agora é um aplicativo completo que permite configurar orçamentos, ajustá-los e calcular automaticamente o orçamento restante para o mês, meus investimentos.

Não é um aplicativo financeiro completo, mas vai me ajudar a controlar meus gastos mensais e ao mesmo tempo aprender mais sobre o framework Django.

## Requirements

 - Python 3.13 ou superior
 - A recent version of Django
 - `UV` (gerenciador de pacotes e projetos Python de alto desempenho escrito em Rust)
